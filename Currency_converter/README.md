# Currency Converter

This Python script allows you to convert currency values from one currency to another using live exchange rate information obtained from the fixer.io API.

## Prerequisites

Before running the script, make sure you have the `requests` library installed, and you should have Python 3 installed on your system.

You can install the `requests` library using the following command:

```bash
pip install requests
```

## How to Run the Script

To run the currency converter, open your terminal and execute the following command:

```bash
python cc.py
```

Follow the on-screen instructions to specify the amount to convert, the source currency, and the target currency. You can also type "SHOW" to see a list of available currencies or "Q" to quit the program.

## Features

- **Live Exchange Rates:** The script fetches live exchange rate information from the fixer.io API.
- **Currency Conversion:** Convert currency values between various currencies.
- **Currency List:** View a list of available currencies.
- **Quit Option:** Quit the program at any time by typing "Q."
