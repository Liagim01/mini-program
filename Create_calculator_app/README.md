# Simple Calculator

This Python script creates a simple calculator application using the `tkinter` library. It allows you to perform basic arithmetic calculations such as addition, subtraction, multiplication, and division.

## Prerequisites

Make sure you have Python 3 installed on your system.

## How to Run the Script

To run the calculator application, open your terminal and execute the following command:

```bash
python calculator.py
```

This will launch the calculator's graphical user interface (GUI).

## Features

- **Input:** Enter mathematical expressions using the provided buttons.
- **Operators:** Use the buttons for addition (+), subtraction (-), multiplication (*), and division (/).
- **Clear:** Clear the input field by pressing the "C" button.
- **Backspace:** Remove the last character from the input field using the "<-" button.
- **Power:** Calculate exponentiation using the "^" button.
- **Zero Division Handling:** The calculator prevents division by zero and displays an alert if you attempt to divide by zero.
- **Quit:** Close the calculator by clicking the "Quit" button.
